# Personal Finance & Budget Monitoring Dashboard

## Overview
This project presents an interactive Power BI dashboard designed to track income, expenses, and savings.  
The dashboard helps monitor personal financial performance and spending behavior using transaction-level data.

## Objectives
- Track total income, total expenses, and net savings
- Monitor savings performance using KPIs
- Compare monthly income vs expenses
- Analyze expenses by category and payment method
- Understand income distribution by source

## Dataset Description
The dataset contains financial transaction records with the following fields:
- Transaction ID
- Date
- Category
- Payment Method
- Amount
- Transaction Type (Income / Expense)
- Account Balance
- Month
- Income Source
- Notes

> Note: The dataset is sample data used for learning and portfolio purposes.

## Key Metrics (DAX Measures)
- Total Income
- Total Expense
- Net Savings
- Saving Rate (%)
- Highest Expense Category

## Dashboard Pages

### Page 1: Financial Overview
- KPI Cards: Total Income, Total Expense, Net Savings, Saving Rate
- Monthly Income vs Expense (Clustered Column Chart)
- Income & Expense Distribution (Stacked Column Chart)

### Page 2: Spending & Income Analysis
- Expense breakdown by category
- Payment method analysis
- Income source distribution
- Interactive slicers (Month, Category, Payment Method)

## Tools & Skills Used
- Power BI
- DAX (CALCULATE, aggregations)
- Data modeling
- Data visualization and storytelling

## Insights
- Expenses exceed income, resulting in negative net savings
- Spending is concentrated in specific categories
- Payment methods influence expense distribution patterns

## Notes
This project focuses on data visualization and analytical storytelling rather than predictive modeling.
